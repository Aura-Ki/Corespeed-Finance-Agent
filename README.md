# Personal Finance Agent
BillSense is a stateful AI finance agent built using Zypher, CoreSpeedâ€™s agent framework.
It allows users to upload financial statements (CSV / Excel / PDF) and interact with the data through natural language, receiving insights, summaries, and actionable recommendations.


## Features
- Smart transaction categorization using AI
- Interactive chat interface for financial queries
- Budget setting and tracking
- Spending predictions based on historical data
- Personalized savings recommendations
- Visual spending reports with charts

## Quick Start
1. Install Deno 2.0+: https://deno.land/
2. Set API key in `.env`
3. Run: `deno task start`
4. Open: http://localhost:8000
5. Upload your bill and start chatting!
6. If you would like, check the demo video here: https://drive.google.com/file/d/19r4eFFI6Jt4huYvkWlmwB9yfYnmeDJit/view?usp=sharing
